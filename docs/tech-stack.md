# Tech Stack

This document lists and briefly describes the technologies used in the Open Hardware Initiative website.

## Main Technologies
- React (UI framework)
- TypeScript (type safety)
- Vite (build tool)
- Tailwind CSS (styling)
- shadcn/ui (UI components)
- Bun (package manager)
- GitHub Actions (CI/CD)
